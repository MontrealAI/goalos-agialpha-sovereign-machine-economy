# PRD — Website Autopilot

## Goal

Generate a public website from `content/site_manifest.json`, run QA, and deploy with GitHub Pages.

## Acceptance criteria

- Every page includes claim boundary.
- Local links are valid.
- Site status JSON is generated.
- Search index is generated.
- QA report is committed.

## Operator benefit

Non-technical users can update the public narrative by editing one JSON file and rerunning one workflow.
