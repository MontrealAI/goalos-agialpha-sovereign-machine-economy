# Proof Settlement API PRD

## Goal

Define a future API for accepting or rejecting work based on Evidence Dockets and Validator Reports.

## Endpoints to design

- create mission;
- submit proof job;
- attach tool trace;
- create evidence docket;
- submit validator report;
- evaluate decision state;
- issue settlement receipt;
- create Chronicle entry;
- register capability package.

## Non-goal

This API blueprint does not authorize user funds, production activation, legal status, or financial settlement.
