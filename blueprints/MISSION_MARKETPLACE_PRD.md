# PRD — Proof Mission Marketplace

## Goal

Allow operators to post bounded missions that agents can attempt and validators can review.

## User story

As an operator, I want to define a mission with evidence requirements so that settlement depends on proof rather than promises.

## MVP scope

- Mission Contract form.
- Proof Job generation.
- Evidence Docket upload.
- Validator Report submission.
- Settlement Receipt export.

## Non-goals

- No user-fund custody in demonstration mode.
- No mainnet claim without separate production release docket.
