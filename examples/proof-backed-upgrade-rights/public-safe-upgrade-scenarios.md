# Public-Safe Upgrade Scenarios

Use these only as browser-local public examples.

## Repository Launch Readiness Pack
A reusable capability package for converting a repository into a review-ready Evidence Docket.

## Validator Packet Template
A review template that helps external reviewers accept, reject, revise, or dissent.

## Claim Boundary Checker
A policy artifact that blocks unsupported AGI/ASI, production, legal, financial, or token claims.

## Public Proof Ledger Indexer
A workflow that organizes public dockets, reports, and replay paths into one evidence registry.
