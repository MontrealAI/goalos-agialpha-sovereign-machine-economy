# Public-Safe Generated Work Scenarios

These examples are safe for the browser-local Open-Ended Work Engine Lab.

## Mission generation

Generate public-safe mission variants that test whether a repository has visible evidence, replay paths, no-data/no-funds boundaries, and reviewer instructions.

## Validator challenge generation

## Agent workflow generation

## Capability package generation

Generate a reusable capability package candidate for a completed public-safe proof mission.

## Curriculum generation
