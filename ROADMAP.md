# Roadmap

## Phase 0 — Public scaffold

- Repository kit.
- Autonomous GitHub Pages site.
- Schemas for proof-bearing work.
- Example mission, docket, settlement, and capability package.
- Claim boundary and non-technical launch guide.

## Phase 1 — Proof kernel

- Stronger schema validation.
- Evidence hash registry.
- Replay log format.
- Validator report aggregation.
- Capability package acceptance gate.

## Phase 2 — Agent and job adapters

- Meta-agentic council adapter.
- AGI Jobs compatibility adapter.
- Node runtime policy adapter.
- Tool trace adapters.
- Risk/cost ledger adapters.

## Phase 3 — Evaluation program

- Real task suite.
- Baselines.
- Delayed outcomes.
- Independent reproduction protocol.
- Public evidence atlas.

## Phase 4 — Settlement simulation

- Non-custodial settlement receipts.
- Validator quorum simulations.
- Capability reuse marketplace simulation.
- No user funds without separate legal, security, and production approvals.
