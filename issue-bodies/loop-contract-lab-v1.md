# GoalOS Loop Contract Lab V1

The Loop Contract Lab adds a browser-local public demo for turning long-running agent loops into reviewable GoalOS loop contracts.

## Review path

- Open `public/loop-contract-lab.html`
- Run loop contract
- Stress weak contract
- Restart from disk
- Toggle gates
- Download artifacts
- Confirm no-data / no-funds / no-wallet boundary

## Boundary

No user data. No user funds. No wallet. No transaction. No network call. No production authority. Human review required.
