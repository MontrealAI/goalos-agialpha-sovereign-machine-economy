# Claims and Evidence Boundary

GoalOS AGIALPHA Ascension — Sovereign Machine Economy is an institutional-grade research, product, and governance scaffold for proof-settled autonomous AI work.

## What the repository claims

It claims to provide:

- A public repository structure for GoalOS-native autonomous work.
- Mission Contract, Proof Job, Tool Trace, Evidence Docket, Validator Report, Settlement Receipt, Chronicle Entry, and Capability Package schemas.
- Example proof artifacts.
- A reference proof kernel.
- A generated GitHub Pages website.
- Repository QA, site QA, claim scanning, artifact digesting, and scorecard scripts.
- Non-technical GitHub Web UI launch guidance.

## What the repository does not claim

It does **not** claim:

- achieved AGI;
- achieved ASI;
- superintelligence;
- empirical SOTA;
- guaranteed security;
- guaranteed returns;
- legal approval;
- tax approval;
- financial advice;
- production authorization;
- user-fund authorization;
- autonomous legal sovereignty;
- mainnet activation;
- external audit completion;
- civilization-scale capability.

## Evidence rule

No strong empirical claim should be published without:

```text
real task
baseline
ProofBundle
Evidence Docket
replay or reproduction path
validator report
cost ledger
risk ledger
delayed-outcome review
human approval for sensitive claims
```

## Publication rule

Public pages must carry the claim boundary. If the claim boundary is removed, publication should fail.


## Strategic evidence boundary

The repository may describe an institutional-scale vision, evidence architecture, proof velocity, and product roadmap. It must not represent private-company valuation, enterprise adoption, revenue, paid pilots, customer commitments, independent validation, or financing outcomes unless those claims are backed by dated source artifacts in the evidence room.

The evidence-to-scale validator treats examples as non-counted placeholders. A strong market-scale public claim requires real artifacts marked as `third_party_verified` or equivalent source-verifiable evidence.
