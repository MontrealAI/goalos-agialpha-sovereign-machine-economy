# GoalOS AGIALPHA Ascension — Sovereign Machine Economy ✨

**The institutional-grade operating surface for proof-settled autonomous AI work.**

GoalOS AGIALPHA Ascension re-implements the earlier **META-AGENTIC α‑AGI 👁️✨ × AGI Alpha Node v0 ⚡️✨ × AGI Jobs v0 (v2) ✨** line as a clean, corporate, public-ready repository for the **Sovereign Machine Economy**: missions, proof jobs, tool traces, evidence dockets, validator reports, governed decision states, settlement receipts, Chronicle entries, and reusable capability packages.

This is designed to feel like a serious institutional product from the first page: elegant public website, non-technical launch guide, claim boundaries, standards, schemas, example proof artifacts, QA reports, operator docs, product blueprints, and a one-file autonomous GitHub Action.

## The thesis

The Transformer scaled intelligence **inside models**.

GoalOS scales intelligence **across governed organizations**.

A model can answer. An agent can act. An institution must prove.

```text
Objective
→ Mission Contract
→ Claims Matrix
→ Proof Jobs
→ Tool Traces
→ Evidence Docket
→ Validator Reports
→ Governed Decision State
→ Settlement Receipt
→ Chronicle
→ Capability Package
```

## Operating law

```text
No proof, no settlement.
No Evidence Docket, no strong public claim.
No eval, no propagation.
No rollback, no release.
No claim boundary, no publication.
Public proof, private data.
```

## One-file GitHub Web UI launch

Create this file in a new GitHub repository:

```text
.github/workflows/goalos-ascension-autopilot.yml
```

Paste the autonomous workflow, commit it, open **Actions**, and click **Run workflow**. The action generates the repository, builds the website, runs QA, commits the artifacts, and deploys GitHub Pages.

## Evidence-to-scale layer

This repository now includes a dedicated evidence-to-scale layer. It separates product vision from counted evidence, requires dated proof artifacts, distinguishes examples from operator-attested records, and blocks strong public scale claims until independent or source-verifiable evidence is present.

The layer includes market evidence schemas, pilot outcome dockets, proof-velocity metrics, public metrics ledgers, strategic evidence scorecards, and an evidence room index. The default examples are intentionally marked as placeholders and do not count as external market proof.

Start with `docs/evidence/EV-001_EVIDENCE_TO_SCALE_STANDARD.md`, then review `reports/strategic-evidence-scorecard.json` after the GitHub Action runs.
