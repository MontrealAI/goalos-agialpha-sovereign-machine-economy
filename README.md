## What this is
GoalOS is a public-alpha proof-native architecture, implementation surface, and website for turning AI work into review-ready evidence, replay paths, claims matrices, validator reports, and governed decision states.

## What this is not
It is not achieved AGI, achieved ASI, empirical SOTA, production certification, mainnet authorization, legal advice, financial advice, tax advice, investment advice, trading advice, wallet support, token distribution, or autonomous production authority.

## Core doctrine
AI creates output. GoalOS creates proof. A model can answer. An agent can act. An institution must prove. Score is advisory. Gates are mandatory. Private intelligence stays private. Public proof becomes inspectable.
