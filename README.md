## What this is

GoalOS is a public-alpha proof institution surface: documents, browser-local demos, schemas, replay paths, and QA scripts for converting AI-assisted work into claim-bounded evidence, governed decision states, and reviewable capability.

## What this is not

It is not achieved AGI, not achieved ASI, not empirical SOTA, not production authority, not legal or financial advice, not a wallet, not a fund-movement system, and not a place to submit private data or funds.

## Core doctrine

AI creates output. GoalOS creates proof. A model can answer. An agent can act. An institution must prove. Do not put intelligence on-chain; put proof of intelligence on-chain. No proof, no evolution. No eval, no propagation. No rollback, no release. Score is advisory; gates are mandatory.

## From Loop to RSI doctrine

RSI is documented as a deterministic invention operating system using TARGET → EMIT → FILTER → ATLAS → TEST-PLAN → EVAL → INSERT → PROMOTE. OMNI is allocation-only and never outcome authority. Move-37 candidates require reproduction, stress testing, persistence gates, and dossier packaging.
