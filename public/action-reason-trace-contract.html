Why this matters

      

        
Action is not authority.
A model may propose. An agent may attempt. An institution must bind every action to permission, evidence, risk, validator status, and rollback.

        
Observation closes the loop.
Expected observation and actual observation must both be recorded. If they diverge, the action must be revised, rolled back, or escalated.

        
External action remains review-bound.
This public demo never performs real external action. It demonstrates the control logic that would make future action auditable.
