Flow academy

Understand the system visually.

GoalOS turns agent activity into institutional proof. These browser-local flowcharts explain the path from a user objective to proof, validation, Chronicle memory, and reusable capability.

Core rule

Not a swarm. An institution.

Agents are roles. AGI Jobs are bounded work. AGI Nodes prepare deterministic runtime handoff. Evidence Dockets make claims reviewable. Human / AGI Node validation governs what can be trusted.

Mission OS flow.

1

AGI Agents

Architect, planner, researcher, builder, verifier, sentinel, docket, and chronicle roles activate only when useful.

2

AGI Jobs

The objective becomes a bounded job with success criteria, constraints, authority, risk, and review path.

3

AGI Nodes

Node Worker / Validator / Sentinel paths prepare deterministic handoff, validation, and safety checks.

4

ProofBundle

Artifacts, outputs, policy decisions, cost, latency, signatures, and replay instructions become reviewable.

5

Evidence Docket

The proof room ties claims to baselines, proof packets, risks, costs, replay, and boundaries.

6

Chronicle

Accepted work becomes institutional memory and a reusable capability candidate.
