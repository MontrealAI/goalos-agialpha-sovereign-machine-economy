Privacy Notice

GoalOS does not want your personal data. The public site is designed to be static, no-account, no-tracking, browser-local, and public-safe.

Plain English: We do not want your data. Do not send personal data or confidential information. The public website is designed as a static, browser-local, no-account, no-tracking surface.

No requested data

  • No account creation on the GoalOS site.
  • No intentional analytics.
  • No newsletter form.
  • No contact form.
  • No payment form.
  • No wallet connection.
  • No model-call data collection.

Do not submit

  • Personal data.
  • Customer or employee data.
  • Private keys, wallet seed phrases, credentials, API keys, secrets.
  • Health, biometric, financial, children’s, criminal, or government-ID data.
  • Confidential, privileged, proprietary, or trade-secret material.

GitHub interactions

GitHub issues, pull requests, comments, stars, forks, and Actions are handled by GitHub. Public submissions to GitHub may be public and governed by GitHub’s own terms and privacy notices. Do not use GitHub public features to submit personal or confidential data.

Data subject and deletion requests

If you accidentally submitted personal data in a public GitHub issue or pull request, edit/delete it through GitHub where available and notify maintainers through GitHub’s appropriate private security/reporting features where possible. We may remove, redact, close, or refuse submissions that appear to contain prohibited data, but we do not promise continuous monitoring.
