# 02 — Research Spine and Canon

The papers are not background. They are the canon.

The repository is now structured as the implementation surface of a three-paper doctrine stack:

```text
AGI ALPHA
→ GoalOS Mission OS
→ Proof-of-Evolution Constitution / AEP‑001
→ Sovereign Machine Economy
```

## AGI ALPHA

**AGI ALPHA** supplies the macro-thesis: the next frontier is not just scaling intelligence inside models, but scaling intelligence across governed organizations.

Its key claim becomes the repository’s institutional foundation:

```text
Transformer = scalable substrate for intelligence inside models
AGI ALPHA = scalable substrate for intelligence across governed organizations
```

The paper maps the organizational substrate:

```text
agents
jobs
validators
tools
memory
markets
settlement
governance
capacity allocation
strategic capability development
```

This repository implements that map as pages, schemas, standards, docs, examples, workflows, and proof surfaces.

## GoalOS Mission OS

**Mission OS** supplies the product operating system:

```text
Objective
→ Mission Contract
→ Autonomous Work
→ Verification
→ Evidence Docket
→ Governed Decision State
→ Action Graph
→ Chronicle
→ Reusable Capability
```

The Mission OS doctrine gives the user-facing promise:

```text
Set the objective. GoalOS runs until proof is done.
AI creates output. GoalOS creates proof.
```

The repository implements this through the Proof Console, Proof Run 001, Evidence Dockets, verifier reports, risk ledgers, claim boundaries, and capability package templates.

## Proof-of-Evolution Constitution / AEP‑001

**AEP‑001** supplies the constitutional layer:

```text
Aim → Act → Prove → Evolve
Commit → Execute → Prove → Evolve
```

It also supplies the strongest law:

```text
Do not put intelligence on-chain.
Put proof of intelligence on-chain.
```

and:

```text
No proof, no evolution.
No eval, no propagation.
No rollback, no release.
```

This repository implements that logic through Proof-Carrying Artifacts, Evidence Dockets, claim boundaries, public/private proof boundaries, Proof Gradient doctrine, Evolution Ledger concepts, and selection gates.
