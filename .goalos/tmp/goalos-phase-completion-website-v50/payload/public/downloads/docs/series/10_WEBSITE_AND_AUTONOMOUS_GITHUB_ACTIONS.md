## Design standard

The current website should not regress from the earlier GoalOS AGIALPHA Ascension aesthetic. The standard is:

```text
cinematic
institutional
grand
proof-first
interactive
local-only by default
claim-bounded
human-review-required
```

## Autonomous publication law

```text
Content Source
→ Generator
→ QA
→ Pull Request / Commit
→ Human Review
→ Publish
```

## Default-deny website boundary

The interactive pages must preserve:

```text
no model call
no wallet
no transaction
no external authority
no production activation
no user-fund authorization
human review required
```

## Why this matters

The website is the public proof interface. Visitors should not only read the thesis. They should experience the proof loop.
