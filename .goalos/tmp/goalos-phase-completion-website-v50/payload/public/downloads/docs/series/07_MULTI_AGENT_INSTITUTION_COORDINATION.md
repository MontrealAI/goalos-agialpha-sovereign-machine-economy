# 07 — Multi-Agent Institution Coordination

We identified the practical control architecture for large multi-agent systems:

```text
proof-gated coordination
```

The core insight is that large multi-agent systems do not coordinate to maximum effect by letting more agents talk. They coordinate to maximum effect when every agent action is converted into bounded work, observable trace, evidence package, validator judgment, risk state, memory update, settlement decision, and reusable capability.

## Coordination law

```text
Maximum coordination ≠ maximum autonomy.
Maximum coordination = maximum verified useful work per unit of risk, cost, latency, and proof debt.
```
