# Procurement Readiness

This repository is not a completed enterprise procurement package, but it includes many of the surfaces that procurement and risk reviewers expect to see.

## Ready surfaces

- license;
- security policy;
- code of conduct;
- contribution model;
- governance document;
- claim boundary;
- threat model;
- artifact schemas;
- QA reports;
- release protocol;
- source lineage;
- public website.

## Still required for production procurement

- legal review;
- data processing terms;
- privacy review;
- security audit;
- dependency review;
- deployment architecture;
- support commitments;
- incident response runbook;
- customer-specific risk assessment.
