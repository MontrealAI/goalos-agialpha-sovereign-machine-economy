# AGI Jobs Dispatch Matrix

What an LLM cannot reliably prove, verify, replay, validate, source, audit, or certify becomes proof debt. GoalOS converts that proof debt into executable AGI Jobs.

## Flow

LLM draft → Proof Debt → AGI Job → ProofBundle → Evidence Docket → Capability Gate → Chronicle hold or promotion.

## Dispatch threshold

Create an AGI Job when missing proof could affect trust, reuse, settlement, publication, payment, safety, compliance, procurement, or future action.
