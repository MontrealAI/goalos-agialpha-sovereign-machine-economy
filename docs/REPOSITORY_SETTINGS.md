# Repository settings

## Social preview

In GitHub: Settings → General → Social preview → Upload `docs/assets/social-preview.svg` or `public/assets/social-preview.svg`.

## GitHub Pages

Settings → Pages → choose the branch and `public/` or configured deployment source used by maintainers. Verify `public/index.html` loads.

## Actions permissions

Settings → Actions → General. Allow workflows needed for QA. Do not grant secrets for docs quality.
