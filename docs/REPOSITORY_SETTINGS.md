# Repository Settings Guide

Open GitHub repository. Use the **About** panel to find the website link. Use **Actions** to run workflows with **Run workflow** when available. Green means success; red means inspect logs. Open workflow artifacts to download reports. Use **Settings > Pages** for GitHub Pages. Use **Settings > Actions > General** to confirm permissions. Use **Settings > General > Social preview** to upload `docs/assets/social-preview.svg`. Use **Releases > Draft a new release** to publish after human review. Use **Issues** and **Pull requests** without private data, secrets, wallets, or funds.
