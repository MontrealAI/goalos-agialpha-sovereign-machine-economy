# Mission OS

Mission Contracts are the unit of governed intention.

## Core idea

GoalOS AGIALPHA Ascension — Sovereign Machine Economy ✨ treats autonomous AI work as an institutional process, not a performance aesthetic. A useful system must show what it was asked to do, what it actually did, what tools were used, what evidence supports the result, what risks were controlled, what it cost, who validated it, and whether the result can safely become reusable capability.

## Practical rule

Every important artifact should answer five questions:

1. What was the bounded objective?
2. What evidence proves the work?
3. What validator or test reviewed it?
4. What decision state did it reach?
5. What rollback or retirement path exists?

## Repository implementation

This repository implements the idea through:

- schemas in `schemas/`;
- examples in `examples/`;
- a reference proof kernel in `src/goalos_ascension/`;
- QA scripts in `scripts/`;
- a generated public website in `public/`;
- release and claim-boundary documents at the repository root.

## Claim boundary

This document is architectural. It does not claim achieved AGI, production activation, external audit, legal approval, financial return, guaranteed security, or user-fund authorization.
