# Glossary

| Term | Plain-language meaning |
|---|---|
| Mission Contract | A written task agreement for AI work. |
| Proof Job | A bounded unit of machine labor with required evidence. |
| Tool Trace | A record of tool actions used to perform work. |
| Evidence Docket | The bundle of proof supporting a claim or result. |
| Validator Report | A review of whether the evidence supports the result. |
| Governed Decision State | The official status of work: draft, blocked, accepted, settled, reusable, retired. |
| Settlement Receipt | A non-custodial record that work was accepted or rejected under stated evidence. |
| Chronicle | The historical ledger of accepted work and system evolution. |
| Capability Package | A reusable versioned asset produced from validated work. |
| Sovereign Machine Economy | A brand/category phrase for operator-controlled, proof-settled machine work; not legal sovereignty or financial promise. |
