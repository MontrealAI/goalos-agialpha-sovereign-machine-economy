# Non-technical guide

GoalOS helps people check whether an AI-assisted result has proof behind it. Think of it as a public-alpha evidence room: claims are listed, evidence is attached, reviewers can replay steps, and high-impact outcomes require human review. Start with the website, Proof Experience Atlas, Demo Registry, and Proof Run 001.
