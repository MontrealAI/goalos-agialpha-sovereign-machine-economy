# Integration Map

This repository re-implements the previous initial iteration as a single GoalOS-native surface.

| Previous layer | New GoalOS layer | What changed |
|---|---|---|
| META-AGENTIC α‑AGI | Meta-Agentic Council | Agent creation and evaluation become governed, docketed, and reversible. |
| AGI Alpha Agent v0 | Agent Foundry | Agents become versioned capability candidates, not unchecked personalities. |
| AGI Alpha Node v0 | Node Runtime | Operator control becomes policy, observability, release-state, and emergency posture. |
| AGI Jobs v0 (v2) | Proof Jobs Ledger | Jobs become bounded work packets with tool traces, acceptance tests, and settlement conditions. |
| GoalOS Mission OS | Mission Contract + Evidence Docket + Governed Decision State | Objectives become verifiable institutional work. |
| Proof Gradient / AEP-001 | Proof-of-Evolution Adapter | System evolution requires mutation evidence, validation, and rollback. |

## System equation

```text
AI Work = Output × Evidence × Validation × Settlement × Reuse
```

If evidence is missing, the work may still be a draft, but it is not settled institutional labor.

## Why this matters

The earlier repositories established agent, node, and job surfaces. This repository makes the connective tissue explicit: the proof substrate that decides when a result is acceptable, reusable, publishable, or blocked.

## Non-technical translation

Think of it as a command center where every AI task leaves a receipt, every receipt has evidence, every evidence package can be reviewed, and every accepted result can become a reusable organizational asset.
