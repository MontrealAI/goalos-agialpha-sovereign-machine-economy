# GoalOS Validation Authority V28 — Human or AGI Node

GoalOS Validation Authority V28 adds a browser-local console where a user can choose the validation authority for a public-safe proof mission.

## Modes

- **Human reviewer**: best for high-impact claims, final publication, sensitive contexts, legal/financial/security posture, or external action.
- **AGI Node validator**: best for deterministic public-safe checks: schema, route fit, boundary, docket completeness, replay readiness, and no-wallet/no-funds constraints.
- **Hybrid**: AGI Node prepares the validation packet; human review closes the loop.

## Outputs

- Validation Certificate JSON
- AGI Node / Human Attestation JSON
- Reviewer Brief Markdown
- Action Graph CSV

## Boundary

No user data. No user funds. No wallet. No transaction. No external call. No production authority. Human review required for high-impact outcomes.
