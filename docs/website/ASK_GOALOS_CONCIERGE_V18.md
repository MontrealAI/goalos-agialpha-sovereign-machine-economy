# Ask GoalOS Concierge V18

Browser-local route assistant for the GoalOS public site.

## What it does

- Adds a floating Ask GoalOS chat window to public HTML pages.
- Answers questions from the local route map.
- Routes users to the right page: Start, Pathfinder, Proof Run 001, Mainnet Contract Atlas, Loop → RSI, Trust, Token Boundary, docs, and all pages.
- Uses no model call, no network call, no analytics, no wallet, and no storage.

## Boundary

No user data. No user funds. No wallet. No transaction. No network call. No production authority. Human review required.
