# GoalOS Validation Command Center V32

## Purpose

V32 gives users a clear way to decide who should validate a GoalOS proof path:

- Human reviewer
- AGI Node validator
- Hybrid: AGI Node precheck + Human final review
- Architect / Validator Council

The user enters a public-safe objective, claim, proof path, Evidence Docket, or mission result. The page generates a validation path, route recommendations, gates, and downloadable artifacts.

## User promise

Human or AGI Node can validate. GoalOS explains which authority is appropriate and why.

## Boundary

This public demo is browser-local. It does not use network calls, wallets, transactions, storage, analytics, or backend model calls.

## Outputs

- Validation Certificate JSON
- Attestation JSON
- AGI Node Handoff JSON
- Reviewer Brief Markdown
- Council Packet Markdown
- Action Graph CSV
