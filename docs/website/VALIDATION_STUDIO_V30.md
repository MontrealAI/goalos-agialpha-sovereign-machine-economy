# GoalOS Validation Studio V30

Human or AGI Node can validate a public-safe GoalOS proof path.

This page provides four authority paths:

- AGI Node validator: deterministic public-safe checks such as schema, route fit, Evidence Docket completeness, claim-boundary wording, replay readiness, token boundary, and site route integrity.
- Human reviewer: public claims, high-impact decisions, legal / financial / security posture, publication approval, procurement, and external-action decisions.
- Hybrid: AGI Node precheck followed by human final review.
- Architect / Validator Council: Loop → RSI governance, Move-37 candidates, strategic dossiers, high-novelty claims, and institutional promotion.

The page is browser-local. It does not collect user data, connect wallets, submit transactions, call a backend, or store user input.
