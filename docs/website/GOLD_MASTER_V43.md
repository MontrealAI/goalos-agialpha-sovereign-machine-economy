# GoalOS Gold Master V43 Runbook

## Purpose

V43 is the public-alpha gold master hardening layer. It preserves all prior GoalOS pages, reinstates missing route surfaces, upgrades the command center, and adds the graduation-readiness runway for future production, empirical SOTA, wallet/transaction, and real-world machine-labor economy claims.

## Install with GitHub Web UI

1. Create `.github/workflows/goalos-gold-master-v43-autopilot.yml`.
2. Upload `goalos-gold-master-v43-pack.zip` to `.goalos/packs/goalos-gold-master-v43-pack.zip`.
3. Commit: `Add GoalOS Gold Master V43`.
4. Run workflow: `GoalOS Gold Master V43 — Public Alpha Gold Master`.
5. Use first-run inputs: commit_changes=true, deploy_pages=true, create_issue=true, create_release=false, release_tag=v0.63.0-gold-master-v43.

## Test

Open `/goalos-gold-master.html`, `/index.html`, `/graduation-control-room.html`, `/production-readiness-gauntlet.html`, `/site-map.html`, `/search.html`, and `/site-health.html`. Confirm all prior pages remain accessible and no wallet/backend/data-collection flow appears.
