# GoalOS AGI Agent Command Center V36

A browser-local public demo for using META-AGENTIC α‑AGI style agents inside GoalOS.

## User promise

Type what you want. GoalOS creates a proof-governed agent mission: Mission Contract, AGI Node handoff, Evidence Docket plan, route cards, reviewer brief, action graph, and next best page.

## Boundaries

No user data. No user funds. No wallet. No transaction. No network call. No production authority. Human review required for high-impact outcomes.

$AGIALPHA public contract identification only. Not available from GoalOS. No sale. No custody. No wallet support. No investment, trading, legal, tax, exchange, bridge, liquidity, or regulatory advice.

## Authority modes

- AGI Node validator: deterministic public-safe checks.
- Human reviewer: high-impact or judgment-heavy review.
- Hybrid: AGI Node precheck plus human final review.
- Council: RSI, Move-37, strategic, or governance-changing claims.
