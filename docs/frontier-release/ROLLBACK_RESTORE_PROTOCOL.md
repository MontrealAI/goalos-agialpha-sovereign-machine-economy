
# Rollback / Restore Protocol

## Rollback triggers

- credible misuse signal
- safeguard bypass above threshold
- access-control failure
- legal or government hold
- validator dissent requiring escalation
- unresolved high-consequence risk

## Restore conditions

- issue reproduced and bounded
- remediation documented
- independent or designated validator review complete
- public summary updated when appropriate
- access tier adjusted
- accountable human authority signs decision state
