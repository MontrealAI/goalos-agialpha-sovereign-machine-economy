
# Public / Private Evidence Model

Frontier release governance must expose enough evidence to support public trust while protecting sensitive details.

## Public layer

- source ledger
- bounded claims
- risk categories
- validator summaries
- access tier decision
- rollback / restore conditions
- limitations

## Private annex

- sensitive cyber evals
- exploit details
- confidential customer records
- protected legal analysis
- classified or controlled information

## Rule

Public proof, private data.
