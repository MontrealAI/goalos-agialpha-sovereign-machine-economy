
# Frontier Release Access Tier Matrix

| Tier | Description | Required evidence | Decision examples |
|---|---|---|---|
| Public | Broad access | public docket, safety summary, monitoring plan | approve public release |
| Limited preview | small vetted set | release plan, access logs, validator summary | approve limited preview |
| Trusted access | qualified partners | vetting record, private annex, controls | approve trusted access |
| Restricted | sensitive capability | risk rationale, remediation plan | restrict access tier |
| Suspended | active hold | incident record, rollback trigger | suspend access |
| Restored | reopened under controls | restore evidence, validator approval | restore with controls |
