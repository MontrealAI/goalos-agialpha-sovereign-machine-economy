
# Validator Seat Model

| Seat | Review focus |
|---|---|
| Cybersecurity | cyber capability, misuse, safeguards, monitoring |
| Model behavior | evals, policy compliance, model limitations |
| Policy / legal | jurisdiction, export-control exposure, institutional constraints |
| Enterprise risk | customer impact, operational continuity, procurement posture |
| Public interest | transparency, civil impacts, public explanation quality |
| Independent reviewer | dissent, replication, external credibility |
