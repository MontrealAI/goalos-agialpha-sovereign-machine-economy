
# GoalOS Frontier Release Room — Operator Guide

## What this is

The Frontier Release Room is an interactive, browser-local proof theatre for release-governance review. It helps visitors understand how GoalOS turns a high-consequence AI release question into an Evidence Docket.

## How to use it

1. Open `frontier-release-room.html`.
2. Select a scenario.
3. Run the proof flight.
4. Inspect each gate.
5. Export the Evidence Docket, review brief, and access matrix.
6. Treat every output as a review package, not an authorization.

## Authority boundary

The console performs no model call, wallet action, API request, transaction, settlement, external deployment, or release approval.

## Best practice

For real use, replace public-source examples with authenticated records, independent validator reports, protected annexes, and an accountable human authorization process.
