# GoalOS AGIALPHA Ascension — Institutional Evidence Release

Use this release when you want the repository to be not only polished, but evidentially disciplined.

## GitHub Web UI launch

1. Create `goalos-agialpha-sovereign-machine-economy`.
2. Set Actions workflow permissions to Read and write permissions.
3. Set Pages source to GitHub Actions.
4. Create `.github/workflows/goalos-ascension-autopilot.yml`.
5. Paste `goalos-autopilot-institutional-evidence.yml`.
6. Run the workflow with `direct_commit_and_publish` and `overwrite_existing_files: true`.
7. Open `reports/strategic-evidence-scorecard.json`.
