# Non-Technical Guide

Open `agent-level-up-lab-v85.html` and press **Simulate Validated Lesson**.

You will see one agent's lesson become a Chronicle candidate and then propagate as a bounded level-up across the agent network. Press **Missing Replay** or **Rejected Verdict** to see why unvalidated output is blocked.

The key idea is simple: agents can learn, but only validated capability can become institutional memory.
