# Learning Propagation Standard

A lesson may influence future missions only when all required gates pass:

1. The lesson is explicit.
2. Proof debt is recorded.
3. AGI Jobs exist for missing evidence.
4. ProofBundle is returned.
5. Replay result passes.
6. Validator verdict approves.
7. Evidence Docket updates.
8. Chronicle candidate is scope-limited and rollback-ready.

If replay or validator verdict is missing, the lesson remains HOLD or QUARANTINE.
