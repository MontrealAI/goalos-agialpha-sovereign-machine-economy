# Proof Run 001 Docket Structure

1. Manifest
2. Claims Matrix
3. Mission Contract
4. Environment
5. Baselines
6. Work Trace
7. Proof Packets
8. Validator Reports
9. Cost Ledger
10. Risk Ledger
11. Replay Instructions
12. Governed Decision State
13. Chronicle Entry
14. Capability Package
15. Public Summary
