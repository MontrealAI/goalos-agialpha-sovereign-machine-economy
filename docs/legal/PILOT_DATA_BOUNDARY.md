# Pilot Data Boundary

No personal, customer, confidential, regulated, proprietary, privileged, or trade-secret data may be submitted through the public website, public GitHub issues, public pull requests, public Evidence Dockets, or public proof demos.

Before any private pilot or enterprise engagement, the parties must execute a separate written agreement covering scope, data categories, roles, confidentiality, security controls, retention, deletion, subprocessors, jurisdiction, audit, incident notice, and review authority.

Absent that written agreement, GoalOS does not accept a data-processing role for private or regulated data.
