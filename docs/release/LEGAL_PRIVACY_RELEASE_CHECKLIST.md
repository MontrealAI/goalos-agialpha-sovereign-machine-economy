# Legal & Privacy Release Checklist

- [ ] Privacy notice present.
- [ ] Terms present.
- [ ] Data boundary present.
- [ ] Token/investment boundary present.
- [ ] Claim boundary present.
- [ ] No intentional analytics scripts.
- [ ] No forms collecting user data.
- [ ] No cookie-setting code intentionally added.
- [ ] No wallet or transaction call.
- [ ] No production authorization language.
- [ ] No achieved AGI/ASI/SOTA claim.
- [ ] No legal/financial/security advice claim.
- [ ] Issue templates warn against personal/confidential data.
- [ ] Pull request template warns against personal/confidential data.
- [ ] Counsel review recommended before commercial or regulated use.
