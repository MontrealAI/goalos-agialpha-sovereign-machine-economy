# Non-Technical Operator Guide

1. Open `chronicle-gate-lab-v80.html` on the website.
2. Read the default objective or type your own.
3. Click **Start Fresh Mission**.
4. Watch the mission create proof debt and AGI Jobs.
5. Click **Simulate Missing Replay** to see the Chronicle gate hold promotion.
6. Click **Simulate Valid ProofBundle** to see Chronicle candidate memory appear.
7. Open the charts to see why future-mission influence remains blocked until validation passes.
8. Download the generated Gate Dossier if you want a JSON record of the run.

Safe default: incomplete proof stays visible as proof debt, not memory.
