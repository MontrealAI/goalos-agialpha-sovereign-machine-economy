# Interface and Chart Guide

The v80 interface includes four local, dependency-free charts.

- **Proof Gate Funnel** shows where an output is blocked: draft, proof debt, jobs, bundle, replay, validator, Chronicle.
- **Chronicle Influence Network** shows that unvalidated outputs have no edge into next-mission planning.
- **Promotion Radar** summarizes replay integrity, validator confidence, claim boundary preservation, risk control, and reuse potential.
- **Proof Economy Thermostat** shows the principle that proof quality raises budget flow while replay failure stops reinvestment.

All charts are rendered locally with inline SVG and JavaScript. No external chart libraries are required.
