# How to Review the Open-Ended Work Engine Lab

Use this page to inspect whether the public demo communicates the core GoalOS doctrine clearly:

1. Run the open-ended cycle.
2. Change the seed objective.
3. Toggle each gate off and verify the decision state changes.
4. Click Stress untethering and verify candidates are quarantined or rejected.
5. Download all artifacts.
6. Confirm the artifacts preserve the seed objective and claim boundary.
7. Confirm no personal data, funds, wallets, transactions, network calls, or production authority are requested.

Expected reviewer verdicts:

- `ENGINE_REVIEW_READY`
- `ADMIT_CANDIDATE_CANARY`
- `QUARANTINE_UNTETHERED`
- `REJECT_UNREPLAYABLE`
- `BLOCK_PRIVACY_BOUNDARY`
