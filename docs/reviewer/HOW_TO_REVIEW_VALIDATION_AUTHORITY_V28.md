# How to Review GoalOS Validation Authority V28

1. Open `public/validation-authority.html`.
2. Validate a public-safe objective with **AGI Node validator**.
3. Validate a high-impact objective with **Human reviewer**.
4. Confirm AGI Node mode blocks or holds high-risk/sensitive requests for human review.
5. Download the certificate, attestation, reviewer brief, and action graph.
6. Confirm no wallet prompt, transaction, backend call, or user-data form appears.
7. Confirm `reports/validation-authority-v28-qa.json` says `passed`.
