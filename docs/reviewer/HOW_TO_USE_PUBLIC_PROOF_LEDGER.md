# How to Use the Public Proof Ledger

1. Open `proof-ledger.html`.
2. Run the 90-second ledger tour.
3. Filter by dockets, reports, pages, or review assets.
4. Inspect the relevant Evidence Docket or report.
5. File accept, reject, revise, or dissent.

Do not submit personal, customer, confidential, regulated, credential, wallet, payment, private-key, seed-phrase, or trade-secret data.
