# How to review Website Experience OS V1

1. Open `/index.html`.
2. Confirm the homepage has a clear hero, path selection, proof journey, evidence/review section, mission/adoption section, and boundary.
3. Press `/` and confirm the command palette opens.
4. Search for `validator`, `docket`, `token`, `action`, and `benchmark`.
5. Open `/website-operating-system.html`.
6. Open `/demo-ecosystem-registry.html`.
7. Confirm the no-data/no-funds boundary is visible.
8. Confirm links are local and no wallet / transaction / network functionality is present.
