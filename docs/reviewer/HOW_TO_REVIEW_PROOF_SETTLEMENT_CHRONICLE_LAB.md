# How to Review the Proof-Settlement Chronicle Lab

1. Open `/proof-settlement-chronicle-lab.html`.
2. Run the settlement cycle.
3. Toggle off `ProofBundle complete` and confirm settlement is blocked.
4. Toggle off `Replay passed` and confirm settlement is blocked.
5. Download the ProofBundle docket and simulated receipt.
6. Confirm the public boundary remains visible: no data, no funds, no wallet, no transaction, no network call, human review required.

Reviewer verdicts should be **accept**, **reject**, or **revise**.
