# How to Review the Action-Reason Trace Contract

1. Open `public/action-reason-trace-contract.html`.
2. Run the action trace.
3. Stress unsafe action.
4. Toggle rollback, validator, boundary, and evidence gates.
5. Confirm the decision state changes.
6. Download the trace contract, action graph, rollback playbook, evidence map, and review brief.
7. Confirm no private data, funds, wallets, transactions, network calls, or production authorization are involved.
