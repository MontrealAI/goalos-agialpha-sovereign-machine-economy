# How to Use the Proof Experience Atlas

1. Open `public/proof-experience-atlas.html`.
2. Run the 90-second tour.
3. Download the Atlas docket.
4. Download the reviewer route.
5. Open Proof Run 001 Execution Room.
6. Review the Evidence Docket, replay checklist, validator packet, and claim boundary.

Do not submit personal data, customer data, confidential data, regulated data, credentials, private keys, seed phrases, or funds.
