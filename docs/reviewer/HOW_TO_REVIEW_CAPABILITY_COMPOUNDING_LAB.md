# How to Review the Capability Compounding Lab

1. Open `public/capability-compounding-lab.html`.
2. Run the compounding cycle.
3. Disable a mandatory gate such as Replay Path or Validator Report.
4. Confirm the Governed Decision State changes to a hold/block state.
5. Download the Evidence Docket and inspect the boundary fields.
6. Confirm no user data, funds, wallet calls, or network calls are requested.
7. Confirm the demonstration does not claim achieved AGI, ASI, production authority, or empirical SOTA.
