# How to Review the Falsification Gauntlet

1. Open `/falsification-gauntlet.html`.
2. Choose each scenario.
3. Run the gauntlet.
4. Toggle off one gate at a time.
5. Confirm the decision state blocks or holds promotion.
6. Download the falsification report and Evidence Docket.
7. Confirm no user data, no funds, no wallet, no transaction, and no network call are requested.

A valid review should mark whether the page makes the claim boundary easier to understand.
