# How to Review Evidence Docket Theatre V2

1. Open `/evidence-docket-theatre.html`.
2. Run docket assembly.
3. Toggle critical docket elements off and watch the decision state change.
4. Download the Evidence Docket JSON.
5. Check that the claim boundary remains visible.
6. Confirm no personal, customer, confidential, regulated, credential, wallet, payment, private-key, seed-phrase, or trade-secret data is requested.
