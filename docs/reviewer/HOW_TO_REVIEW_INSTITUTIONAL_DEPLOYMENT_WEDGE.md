# How to Review the Institutional Deployment Wedge

Open:

`https://montrealai.github.io/goalos-agialpha-sovereign-machine-economy/institutional-deployment-wedge.html`

Review checklist:

1. Confirm the page is readable and not blank.
2. Run the deployment wedge.
3. Toggle canary, rollback, replay, baseline, and boundary gates.
4. Confirm the decision state changes.
5. Confirm downloadable artifacts include the scenario, objective, gates, and boundary.
6. Confirm no user data or funds are requested.
7. Confirm the page remains clear in Executive and Technical modes.

Expected public-alpha boundary:

- no user data
- no user funds
- no wallet
- no transaction
- no network call
- no production authority
- human review required
