# How to review the Proof-Carrying Artifact Foundry

Reviewers should check:

1. Does the artifact have a clear class and scope?
2. Does it include proof history?
3. Did an eval pass?
4. Was it compared against a baseline?
5. Does it include rollback readiness?
6. Is the scope authorized?
7. Has the challenge window cleared?
8. Is the replay path present?
9. Is the public/private proof boundary preserved?
10. Does the page avoid user data, funds, wallet, transaction, and network calls?

Verdict options:

- Accept for public-alpha review.
- Hold pending proof.
- Hold pending rollback.
- Hold pending replay.
- Reject privacy-boundary failure.
- Reject restricted scope.

The page is a demonstration of proof-governed artifact promotion, not production authorization.
