## Boundary

No user data. No user funds. No wallet. No transaction. No network call. No production authority. Human review required.

Do not enter personal, customer, confidential, regulated, credential, wallet, payment, private-key, seed-phrase, health, children’s, privileged, trade-secret, or proprietary data.
