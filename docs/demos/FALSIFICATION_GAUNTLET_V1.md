# GoalOS Falsification Gauntlet V1

**Public page:** `public/falsification-gauntlet.html`

This browser-local demo demonstrates the GoalOS rule:

> A system that cannot fail cannot prove.

The page lets users run a public-safe claim through baselines, replay, cost/risk ledgers, validator notes, privacy boundaries, claim boundaries, delayed-outcome planning, and human-review requirements.

## Boundary

- No user data.
- No user funds.
- No wallet.
- No transaction.
- No network call.
- No production authority.
- Human review required.

## Demonstrated loop

Claim → Claims Matrix → Baselines → Replay → Cost/Risk Ledger → Validator Notes → Claim Boundary → Falsification State → Governed Decision State.
