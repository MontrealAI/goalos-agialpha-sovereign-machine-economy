# GoalOS From Loop to RSI Sovereign Console V2

**Build the governance institution first.**

This browser-local public demo shows how a restartable agent loop becomes deterministic RSI governance:

```text
Loop Contract → Disk State → Drift Sentinel → ECI Ledger → Baseline Gate → Move-37 Dossier → Architect/Validator Council → Governed Decision State
```

The page is designed for non-technical users and advanced reviewers. It demonstrates that RSI is not a novelty mill, not a black-box authority, and not unrestricted recursive self-improvement. Exploration is allowed; outcome authority is earned through evidence, baselines, replay, persistence, and human/institutional review.
