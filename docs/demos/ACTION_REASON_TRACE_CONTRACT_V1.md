# GoalOS Action-Reason Trace Contract V1

**Every action must carry a reason.**

This browser-local public demo shows how GoalOS binds any external-action request to a reviewable trace: objective, permission scope, reason, expected observation, actual observation, validator status, cost/risk ledger, rollback pointer, evidence pointer, and human review.

It is not a wallet, transaction system, backend service, analytics funnel, or production authorization tool.

Boundary: no user data, no user funds, no wallet, no transaction, no network call, no production authority; human review required.
