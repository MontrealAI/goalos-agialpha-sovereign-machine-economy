# GoalOS Loop Bottleneck Observatory V1

**The bottleneck always moves.**

This browser-local public demo shows how GoalOS turns long-running agent loops into governed proof loops. It complements the Loop Contract Lab and Loop Flight Recorder by making the next bottleneck visible: contract, role separation, state, traces, restartability, evaluator independence, subjective rubric, harness overhead, or public/private proof boundary.

## Public lesson

A prompt produces output. A loop produces governed work. A GoalOS loop produces proof.

## Demo lifecycle

Objective → Role Contracts → Disk State → Trace Reading → Restart → Evidence Docket → Bottleneck Report → Governed Decision State

## Safety boundary

No user data. No user funds. No wallet. No transaction. No network call. No production authority. Human review required.
