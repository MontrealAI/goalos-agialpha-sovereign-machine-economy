# GoalOS Real-Task Benchmark Bridge V1

**Public thesis:** demos explain the architecture; real-task benchmark bridges make claims empirical.

This browser-local page demonstrates how a GoalOS claim moves from public-alpha architecture toward empirical evidence:

```text
Task family
→ real-task manifest
→ equal-budget baselines
→ ProofBundle / Evidence Docket
→ replay path
→ cost/risk ledger
→ validator report
→ delayed-outcome plan
→ external reviewer path
→ governed decision state
```

The page is deliberately conservative. A strong claim is blocked when baselines are missing, replay is absent, ledgers are incomplete, or external review is unavailable.

## Boundary

No user data. No user funds. No wallet. No transaction. No network call. No production authority. Human review required.
