# GoalOS Proof-Carrying Artifact Foundry V1

**Thesis:** the artifact earns authority.

This public browser-local demo shows how a raw output becomes a Proof-Carrying Artifact: a scoped, immutable, evidence-bearing, rollbackable object that can influence future work only after hard gates pass.

## What it demonstrates

- A model can answer; an agent can act; an institution must decide what may be reused.
- The central reusable object is not an agent or a model; it is a Proof-Carrying Artifact.
- Score is advisory; gates are mandatory.
- No proof, no evolution. No eval, no propagation. No rollback, no release.
- Private intelligence stays private; public proof carries hashes, attestations, replay paths, selection status, and claim boundaries.

## Browser-local boundary

No user data. No user funds. No wallet. No transaction. No network call. No production authority. Human review required.
