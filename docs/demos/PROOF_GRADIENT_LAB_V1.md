# GoalOS Proof Gradient Lab V1

A browser-local public demonstration of the GoalOS / AEP-001 doctrine: Aim → Act → Prove → Evolve.

## Core lesson

Score is advisory. Gates are mandatory. A candidate capability may evolve only when proof, evaluation, rollback, canary, scope, and challenge gates pass.

## Boundary

No user data. No user funds. No wallet. No transaction. No external action. Human review required.
