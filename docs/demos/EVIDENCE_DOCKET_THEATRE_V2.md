# Evidence Docket Theatre V2

A browser-local public demonstration of the GoalOS proof room. It teaches the distinction between a marketing claim, an Evidence Docket, and a Governed Decision State.

Core law: a proof page is not a marketing page. It must show claims, baselines, proof packets, evaluator notes, risk/cost ledgers, replay path, claim boundary, and human-review status.

Boundary: no user data, no user funds, no wallet, no transaction, no external action, human review required.
