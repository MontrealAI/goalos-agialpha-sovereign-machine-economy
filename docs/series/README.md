## Institutional Document Series

This folder contains a polished document series for the public repository, website, board-level explanations, operator onboarding, and enterprise trust review.

The series is intentionally written for non-technical readers first. Technical reviewers still get the structure they need: schemas, proof flow, validation gates, governance boundaries, security posture, repository paths, pilot metrics, and publication controls.

## Recommended placement in the repository

Place the files at:

```text
docs/series/
```

Then link `docs/series/00_SERIES_INDEX.md` from the main repository `README.md`, `START_HERE.md`, and the generated website.
