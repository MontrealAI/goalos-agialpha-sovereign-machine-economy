# Operating Metrics

GoalOS should be measured by proof quality, not by theatrical autonomy.

| Metric | Meaning |
|---|---|
| Mission completion rate | percentage of missions reaching review state |
| Docket completeness | percentage of required evidence fields present |
| Validator agreement | consistency of validator conclusions |
| Contradiction closure | how quickly contradictions are resolved or bounded |
| Reuse rate | accepted capabilities reused in later missions |
| Rollback readiness | whether releases have rollback paths |
| Claim correction latency | how quickly unsupported claims are removed |
| Cost trace coverage | percentage of work with cost ledger entries |
| Source provenance coverage | percentage of factual claims linked to provenance |
| Delayed outcome review | whether outcomes remain valid after time passes |
