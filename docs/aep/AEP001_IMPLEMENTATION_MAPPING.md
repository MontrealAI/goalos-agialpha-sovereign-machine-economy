# AEP-001 Implementation Mapping

AEP-001 defines Aim → Act → Prove → Evolve and the protocol object lifecycle Commit → Execute → Prove → Evolve.

This repository implements the reference public-alpha version through:

- `GoalOSCommit`
- `RunCommitment`
- `ProofPacket`
- `EvalAttestation`
- `SelectionCertificate`
- `EvolutionLedgerEntry`
- `ProofCarryingArtifact`
- `EvidenceDocket61`
- `ProofGradient`

The on-chain layer is represented only as schema and simulated proof objects. No chain transaction, wallet interaction, or mainnet authority is created by this reference implementation.
