# GoalOS Multi-Agent Institution Experience V6

**Not a swarm. An institution.**

Large multi-agent systems coordinate to maximum effect when they become proof-governed institutions.

This implementation upgrades `public/multi-agent-institution.html` into a browser-local institutional proof lab for both non-technical and advanced users.

## What users can do

- Choose a public-safe mission preset.
- Switch between Executive and Technical modes.
- Tune proof controls.
- Run a proof flight.
- Compare a raw swarm against a proof-governed institution.
- Inspect proof gates.
- Download a local Evidence Docket.
- Download a review brief.
- Download a capability package.

## Boundary

No user data. No user funds. No wallet. No transaction. No network/API calls from the demo. No production authority. Human review required.

## Why this matters

A swarm produces activity. An institution produces reviewable state: mission, roles, tools, evidence, validators, risk ledger, claim boundary, governed decision state, Chronicle memory, and reusable capability.
