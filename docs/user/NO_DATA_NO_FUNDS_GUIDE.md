# No Data / No Funds Guide

GoalOS public channels are public-safe only.

## Do not submit
Do not submit personal, customer, confidential, regulated, credential, wallet, payment, trade-secret, private-key, seed-phrase, health, children’s, privileged, or proprietary data through the public website, GitHub Issues, Discussions, pull requests, demos, Evidence Dockets, or public proof runs.

## Do not send
Do not send ETH, $AGIALPHA, stablecoins, NFTs, private keys, seed phrases, approvals, wallet signatures, payment data, or anything with monetary value.
