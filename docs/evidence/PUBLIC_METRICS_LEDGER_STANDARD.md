# Public Metrics Ledger Standard

## Purpose

The public metrics ledger publishes the metrics that can be supported without exposing private data. It separates public proof from confidential records.

## Recommended public metrics

- Missions completed.
- Evidence Dockets created.
- Validator reports completed.
- Release gates passed.
- Capabilities reused.
- Average time from objective to docket.
- Average verification time.
- Reproducibility rate.
- Evidence completeness score.
- Number of independent reviews.

## Private metrics

Customer names, contracts, revenue records, private communications, and sensitive operational data should remain private unless explicit publication approval exists. The public ledger can publish aggregates and evidence hashes.


## Claim discipline

This document is an operating standard. It does not claim financing, market valuation, production authorization, legal approval, user-fund authorization, revenue, enterprise adoption, or independent audit completion. Counted evidence must be dated, source-verifiable, and stored in the evidence room.
